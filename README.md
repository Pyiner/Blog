# Yiner in Python #

用Python Django实现的个人博客系统，后台用的Django Admin，留言托管在多说上，图片存储在百度云存储。

重要实现的功能有：

- 文章显示
- 分页
- 友情链接
- 开源项目
- 最热文章
- RSS输出
- 文章定时同步到百度云存储
